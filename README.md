# Secure Quantum Communication Protocol (BB84 Simulation)

This project demonstrates the working of the BB84 Quantum Key Distribution protocol using Python. The goal of the project is to simulate secure communication and detect potential eavesdropping attempts in a quantum cryptographic system.

## Technologies Used
- Python
- Basic Cryptography Concepts
- Quantum Key Distribution Logic

## Features
- Simulation of photon polarization states
- Random basis selection for sender and receiver
- Secure key generation
- Detection of eavesdropping attempts
- Quantum Bit Error Rate analysis

## Project Description
The BB84 protocol is one of the first quantum cryptography protocols designed to allow two parties to generate a shared secret key securely.

This simulation demonstrates how secure keys are exchanged between two parties while detecting interception attempts by a third party.

## Learning Outcomes
- Understanding quantum cryptography fundamentals
- Secure key exchange principles
- Detection of man-in-the-middle attacks
